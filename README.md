# 架构 struct
这个游戏引擎目前有
渲染类，对象类，摄像机类
实现了基本渲染功能

渲染类是一个渲染器，我们可以添加渲染对象到渲染队列，然后在渲染时期按照添加对象的顺序依次渲染（覆盖）
渲染对象目前有1.动画类2.动画状态机3.动画组
具体功能在代码中注释很详细，在这里只是简单介绍
动画类用于播放单个动画；动画状态机用于管理动画类，但是一次只能播放一个动画；动画组类将多个动画（动画状态机或动画类）融合成为一个，配合shader能够实现简单的透明度混合功能
